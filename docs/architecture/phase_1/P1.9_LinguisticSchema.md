## 4. Core Logic / Math

- **Herb interface:** `aliases?: string[]`, `linguistics?: { tonal_pinyin?, jyutping?, hokkien? }`
- **searchHerbs:** Matches query against pinyin_name, common_name, english_name, or any `aliases[]` entry; optional chaining for safe fallback
- **displayPinyin:** Uses `linguistics.tonal_pinyin` if present, else `pinyin_name`
- **getMatchedAlias:** Returns alias string when query matches; drives "(Alias match: X)" in dropdown
