## 4. Core Logic / Math

- **Read:** Load `data/output/seed_herbs.json` (SymMap schema: `properties[]`, `meridians[]`, `english_name`, `chinese_name`)
- **Temperature extraction:** Scan `properties[]` for keywords (Cold, Cool, Warm, Hot, Calm, Neutral, Slightly/Very variants); first match → `properties.temperature`; default `"Neutral"` if none
- **Flavor extraction:** All `properties[]` terms not matched as temperature → `properties.flavor` (e.g. Pungent, Bitter, Sweet, Toxic)
- **Meridians:** Top-level `meridians[]` → `properties.meridians` unchanged
- **common_name:** `english_name` or fallback `chinese_name`
- **actions:** Default `["Harmonizes systemic function"]`; LLM enrichment planned
- **contraindications:** Default `""`
- **Write:** Overwrite `src/data/seed_herbs.json` with mapped array
