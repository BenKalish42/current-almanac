## Test Suites

### `src/utils/heuristicRater.spec.ts` (The Math Test)

- Mocks two herbs: one Warm/Sweet (Qi-focused), one Cold/Bitter (Jing-focused)
- Passes them into `calculateFormulaRating()` with empty `herbRoles`
- Asserts:
  - `jingScore` = 50 (scaled from raw Jing contribution)
  - `qiScore` = 100 (scaled from raw Qi contribution)
  - `shenScore` = 0
  - `primaryEffect` = "Primary Qi Tonic"

### `src/stores/alchemyStore.spec.ts` (The State Test)

- Initializes Pinia via `setActivePinia(createPinia())` in `beforeEach`
- Creates a minimal mock `Herb` and calls `addHerbToFormula(herb)`
- Asserts `activeFormula.length === 1`
- Calls `clearFormula()`
- Asserts `activeFormula.length === 0`
