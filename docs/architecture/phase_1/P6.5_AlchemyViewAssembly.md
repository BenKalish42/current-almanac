## 4. Core Logic / Math

- **Layout:** CSS Grid `grid-cols-1 lg:grid-cols-2`, stacking to single column on mobile
- **NPDI banner:** Conditional `v-if="alchemyStore.npdiWarnings.length"`; amber/red styling for warnings
- **JingBattery:** Dummy props `yinLevel=70`, `yangLevel=45` (reactive placeholder for future state)
- **Cultivation Analysis:** Binds `formulaCultivationRating` from store; displays jingScore, qiScore, shenScore, primaryEffect
- **Spacing:** `gap-6`, `p-4`; Daoist dark palette (`daoist-surface`, `daoist-elevated`, etc.)
