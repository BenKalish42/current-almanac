# Current Almanac

Vue 3 + TypeScript + Vite application for web, Capacitor mobile shells, and Electron desktop packaging.

## Release targets

- Web deploys continue through Netlify.
- iOS and Android native shells live in `ios/` and `android/`.
- Desktop packages are built with Electron Forge.
- One-click coordinated releases run from the `Release Everywhere` GitHub Actions workflow.

See `docs/deployment/app-store-desktop-release.md` for required secrets and release steps.
